Softwares

GMPE-package

R Jun 2021
This package contains 11 commonly used Ground Motion Prediction Equations (GMPE) for both crustal and subduction earthquakes.

hvsrProc-package

R Apr 2021
This package was developed for processing and calculating HVSR from noise-based and earthquake-based time series data.

HVSR Database Access and Analysis

Python Mar 2021
A project created on the Designsafe. It provides a Jupyter Notebook that interacts with the HVSR database to perform two operations that are anticipated to be useful for research of site response effects in which HVSR is used for site characterization: 1) identify if a site has HVSR peaks; and 2) for sites where peaks are present, establish attributes of the peaks such as peak frequency and amplitude.

RCTC-package

R Oct 2017
The package was developed for Rotated Combination of Two-Component (RCTC) Ground Motions calculation.
